Code For Belgium Community Images
==================================

This folder contains all images for the Code For Belgium community.

REQUIRED IMAGES:
---------------
1. card.webp - Image shown on the community cards on the homepage
2. hero.webp - Large banner image at the top of the detail page
3. link-cover.webp - Image shown in the contact section (website preview)

HOW TO UPDATE IMAGES:
--------------------
1. Replace the image file with your new image (keep the same filename)
2. Make sure the new image is in .webp format
3. Refresh your browser to see the changes

TO ADD EVENT ALBUMS:
-------------------
1. Add your event photos to this folder
2. Open src/data/communities.js
3. Find the Code For Belgium community section
4. Add a new album entry with your image paths
   Example: '/images/code-for-belgium/event-photo.webp'
