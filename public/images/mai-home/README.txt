MAI-HOME Community Images
=========================

This folder contains all images for the MAI-HOME community.

REQUIRED IMAGES:
---------------
1. card.webp - Image shown on the community cards on the homepage
2. mai-home-hero.webp - Large banner image at the top of the MAI-HOME detail page
3. mai-home-cover.webp - Image in the "What is MAI-HOME?" section
4. mai-home-qr.webp - WhatsApp QR code shown in the contact section

HOW TO UPDATE IMAGES:
--------------------
1. Replace the image file with your new image (keep the same filename)
2. Make sure the new image is in .webp format
3. Refresh your browser to see the changes

TO ADD NEW EVENT PHOTOS:
-----------------------
1. Add your new images to this folder
2. Open src/data/communities.js
3. Find the MAI-HOME community section
4. Update the albums array with the new image paths
   Example: '/images/mai-home/your-new-image.webp'
